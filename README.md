# Spring Boot + Vue 电影购票系统

## 一、项目结构

- 网上购票
- 后台管理
- 客服模块

## 二、环境介绍

- 后端：SpringBoot + SpringSecurity + MyBatis-Plus
- 前端：Vue + ElementUi
- 数据库：MySQL

## 三、关于项目

- api: Spring Boot项目代码
- vue-admin: 网站管理后台代码
- vue-app: 网站前台代码
- vue-worker: 网站客服代码
